# Promo Assets

This directory contains promotional images for the GitHub Code Wiki Button extension.

## Required Images

When publishing to Chrome Web Store or other browser stores, you'll need:

### Chrome Web Store
- **Store Icon**: `Chrome-Webstore-Icon_128x128.png` (128x128px)
- **Screenshot**: `Screenshot_1280x800.png` (1280x800px)
- **Marquee**: `Promo-Image-Marquee_1400x560.png` (1400x560px)
- **Small Promo**: `Promo-Image-Small_440x280.png` (440x280px)

### Firefox Add-ons
- **Screenshots**: Multiple screenshots showing the extension in action

### Microsoft Edge Add-ons
- Similar to Chrome Web Store requirements

## Creating Images

You can use tools like:
- Figma / Sketch / Adobe XD for design
- Screenshots from actual browser usage
- Image editing tools (Photoshop, GIMP, etc.)
