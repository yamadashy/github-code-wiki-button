Αυτή η επέκταση προγράμματος περιήγησης προσθέτει ένα βολικό κουμπί "Code Wiki" στις σελίδες αποθετηρίων του GitHub, επιτρέποντάς σας να έχετε γρήγορη πρόσβαση στην τεκμηρίωση του αποθετηρίου στο Code Wiki της Google.

🛠️ Χαρακτηριστικά:
- Ομαλή ενσωμάτωση με τη διεπαφή του GitHub
- Πρόσβαση με ένα κλικ στο Code Wiki για οποιοδήποτε αποθετήριο GitHub
- Καθαρός και μινιμαλιστικός σχεδιασμός ακολουθώντας τα πρότυπα διεπαφής του GitHub
- Λειτουργεί σε όλες τις σελίδες αποθετηρίων του GitHub

🚀 Τρόπος χρήσης:
1. Εγκαταστήστε την επέκταση
2. Πλοηγηθείτε σε οποιοδήποτε αποθετήριο GitHub
3. Κάντε κλικ στο κουμπί "Code Wiki" στην κεφαλίδα του αποθετηρίου
4. Θα ανακατευθυνθείτε στην αντίστοιχη σελίδα του Code Wiki

⚠️ Σημείωση:
Αυτή είναι μια ανεπίσημη επέκταση και δεν συνδέεται με το GitHub ή την Google. Απλώς παρέχει έναν βολικό τρόπο πλοήγησης μεταξύ των αποθετηρίων GitHub και των αντίστοιχων σελίδων Code Wiki.

🔒 Απόρρητο:
- Καμία συλλογή δεδομένων
- Καμία παρακολούθηση
- Δεν απαιτείται έλεγχος ταυτότητας
- Απαιτείται μόνο πρόσβαση στο github.com για την προσθήκη του κουμπιού
